Medibot is an AI-powered chatbot designed to enhance hospital patient experience by automating routine interactions. Built using Amazon Web Services (AWS), the chatbot enables patients to:

  • Check appointment availability
  • Book or cancel appointments
  • Get instant answers to FAQs

Core Technologies Used

  • Amazon Lex: Builds the natural language interface, allowing patients to interact via voice or text.
  • AWS Lambda: Powers backend logic to handle user intents.
  • Amazon CloudWatch: Monitors logs and metrics for performance and stability.

Key Features & Benefits

  • Conversational AI: Reduces need for human intervention.
  • Scalable & Serverless: Built on AWS for low operational overhead.
  • Secure Data Handling: Aligns with healthcare privacy practices.

  • Reduced Admin Load: Automates routine front-desk tasks.
  • Efficient Resource Use: Enhances scheduling and operations.
